1.Download wifi.exe
2.Give administrator privileges by right clicking exe file and choosing option "Run as administrator".
3.add the path to environment variables.
4.Open commandprompt.

Note: Only supported for laptops with hotspot supporting wifi modules and also same in case of PCs.

How to use.
1. For setting wifi,type 'wifi START' followed by 'ssid' and 'password'.
2. To enable wifi, type 'wifi ON'.
3. To disable wifi, type 'wifi OFF'.
